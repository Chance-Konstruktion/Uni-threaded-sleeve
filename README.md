# Uni-threaded-sleeve

Vollständiger parametrischer Gewindemuffen / Sleeve Generator für Blender.

**Benötigt:** [Uni-threaded-rod](https://github.com/Chance-Konstruktion/Uni-threaded-rod)

## Features
- Alle Gewindetypen (Metrisch, Trapez, Rohr, UNC, NPT etc.)
- Presets (Standard, Reparatur, Rohr, Heavy Duty, Flansch...)
- Rund oder Sechskant Außenform
- Flansch ein-/beidseitig
- Clearance für Reparatur-Sleeves
- Voll integriert mit Uni-threaded-rod Engine

## Installation
1. Uni-threaded-rod zuerst installieren
2. Dieses Add-on installieren
3. Beide aktivieren

Tab erscheint im Sidebar unter **Uni-threaded-sleeve**.
